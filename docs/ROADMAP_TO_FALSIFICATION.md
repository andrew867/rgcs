## Ordering and gating

1. **Calibration gates before any hypothesis test:** instrument phase-drift budget (A-21, KOS-13); coherence baseline b_w and detection-limit test (H-14 prerequisite); correction-ledger stability (LT-13 consequence): repeated δ_k measurements stable before any ε_R^(f) is interpreted (gates H-02).
2. **Identifiability gates (M.60):** γ_n from free ringdown before G_n fits (H-09); κ_χ reported instead of (v_χ, R_χ) unless H-06a supplies an independent prior; η fixed or two-loading protocol before ΔM_H is quoted (H-08).
3. **Statistical discipline:** all tests pre-registered with α, n, and analysis windows; look-elsewhere effects controlled by scrambled-label nulls (M.60 item 4); no post-hoc bin or window changes (policy §3.2).
4. **Vocabulary discipline:** no result may be described with dark-sector, BEC/condensate, or quantum-cosmology vocabulary (policy §2; QA gate for Agent 08).

## What is NOT a hypothesis here

- **f_0 = 4096 Hz significance** — Source claim; RGCS tests specimens built on the ladder (H-01a) but does not test "4096 is special"; no experiment in this roadmap can confirm numerological significance (RG-17 containment).
- **Established mathematics** (eigenvalue algebra, circular statistics, coherence definition) — verified by golden-value unit tests (ledger Part E), not experiments.
- **Source presets** (20 Hz/15 V, 1496 Hz, 46/46/184, coil points) — Source claims; they parameterize protocols but carry no truth value of their own.
