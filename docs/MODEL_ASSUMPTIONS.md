# RGCS v2 — Model Assumptions Register

**Author:** Sub-Agent 02
**Date:** 2026-07-14
**Rule:** Every assumption is numbered A-xx, states where it enters (equation IDs), its classification, and the concrete consequence if it is violated. Downstream agents cite assumptions by ID. An assumption is not a hypothesis: hypotheses (H-xx) are falsifiable claims under test (`ROADMAP_TO_FALSIFICATION.md`); assumptions are modeling conditions whose violation changes which equations are valid.

| ID | Assumption | Enters at | Classification | Consequence if violated |
|---|---|---|---|---|
| A-01 | The 1D axial half-wave model applies to a faceted, tapered, double-terminated crystal: the fundamental extensional mode behaves like a free–free uniform bar to first order. | M.8, M.9, M.12, M.39 | Hypothesis-grade assumption (RG-01) | All ladder lengths and harmonic classes are systematically biased; taper and caps shift the true mode by an unmodeled δ_geometry; the shaft-midpoint node prior x_g loses its derivation. Mitigation: FEA or measured mode shapes replace the 1D estimates; the ladder becomes a naming convention only. |
| A-02 | A single effective scalar wave speed v_L with relative uncertainty u_v = 0.05 adequately brackets quartz anisotropy until specimen orientation is measured. | M.10–M.12, M.15 | Derived (declared budget) over Established anisotropy physics | If a specimen's propagation direction is far from Z, the true speed can sit at the band edge or outside it; harmonic classes flip. Consequence is contained by the set-valued class 𝒩 (M.12); if measured modes fall outside the band, u_v was too small — widen and re-issue all predictions. |
| A-03 | Linear elasticity, small strain, amplitude-independent ω_n and Q_n over the operating drive range. | M.13, M.23–M.28, M.46 | Established (regime assumption) | Frequency pulling with amplitude (Duffing-type) invalidates fixed-ω fits; ε_R^(f) becomes drive-dependent. Detection: f vs drive-power sweep; if slope ≠ 0, add cubic stiffness term before interpreting ε. |
| A-04 | Modal separability: modes are well separated (spacing ≫ Γ) except for the deliberately modeled near-degenerate pairs/sets in 𝐇. | M.23–M.28, M.46 | Established (regime assumption) | Eigenvalue fits absorb unmodeled neighbors into biased g_nm; splitting misattributed. Detection: residual structure in lineshape fits; remedy: enlarge 𝐇. |
| A-05 | Slowly-varying envelope: Q_n ≫ 1 so amplitude/phase evolve slowly relative to the carrier (validity of the rotating-frame reduction). | M.46–M.49 | Established | For Q ≲ 10 the envelope ODEs misestimate rates by O(1/Q); use full linear coupled-mode validation harness (Module 10, alternative 1) instead. |
| A-06 | A closed compact path supporting propagating waves at speed v_χ exists in the resonator (existence of χ). | M.13–M.17, M.50 | Hypothesis-grade assumption (RG-05) | The compact spectrum is meaningless as physics (remains a curve-fitting family); H-01 model comparison is the arbiter. |
| A-07 | Mean-per-turn averaging: R_χ^(s) = ℓ_3D/(2πT) treats all turns as equivalent although each turn's length contracts by ≈ 1/q. | M.36 | Hypothesis (definition choice, RG-09) | If the physical path weights turns unequally, the prior is biased by up to the per-turn spread (factor q^(T−1) across turns); per-turn model M.37 must be compared (M.60 item 2). |
| A-08 | Parity is measurable: two-sensor phase or FEA mode shapes suffice to assign odd/even symmetry classes unambiguously. | M.16–M.17 | Established (methodology) | Without measurable parity the families are cosmetic filters (the pre-v2 defect, LT-Δ1); parity-dependent claims must then be suspended. |
| A-09 | Human/fixture loading acts as pure added modal mass with participation fraction η = 0.5, without stiffness or damping change. | M.42–M.44 | Hypothesis-grade assumption (RG-10) | If contact adds stiffness, k_H > 1 is possible and ΔM_H is meaningless (flagged by the `requires stiffness/mode correction` branch); if it adds damping, Q drops and linewidth-based classes (M.20) shift. η unmeasured ⇒ ΔM_H known only up to scale (identifiability register, M.60). |
| A-10 | Correction terms δ_k are small (|Σδ| ≤ 0.02), mutually independent, and additive to first order. | M.22, M.45 | Derived (ledger form) | Cross-terms δ_i·δ_j become significant; the ledger misestimates f_meas and u(ε_R); switch to the nonlinear/Bayesian model (M.59) — the ledger is bookkeeping, not physics. |
| A-11 | Stationarity within the coherence window w: signal statistics change slowly relative to w. | M.53, M.56 | Established (signal processing) | Transients smear 𝒞_w and Σ_φ; window-length sensitivity analysis (mandatory per KOS-07) detects it; report 𝒞 for multiple w. |
| A-12 | The three sensor channels are orthogonal, aligned to declared specimen axes, and gain/phase-calibrated against each other. | M.51–M.53 | Established (calibration requirement) | Channel cross-talk masquerades as phase locking (Σ_φ biased low) or miscalibration as anisotropy (σ_φ² biased high). Control: instrument-only characterization precedes sample runs (KOS-13 pattern). |
| A-13 | Measurement noise is additive, Gaussian, independent across observations, with known/estimated s_i. | M.59–M.61 | Established (statistical model) | Heavy tails or correlated noise invalidate stated p-values and AIC gaps; bootstrap fallback (M.59) and artifact register (KOS-14) are the mitigations. |
| A-14 | The drive does not alter material parameters (no heating drift, poling, or damage) within a session beyond the δ_T, δ_drive ledger terms. | M.45, M.46 | Established (regime assumption) | Session-order effects appear; randomized run order and pre/post free-mode checks detect; affected runs quarantined. |
| A-15 | Zero-mode handling follows M.17: odd family never has n = 0; even family's n = 0 governed by `include_zero_mode`; f_b = 0 ⇒ n = 0 excluded. | M.16–M.17 | Derived (decision, closes D-14) | Silent inclusion/exclusion of a base mode changes mode counting by one everywhere — off-by-one n assignments corrupt κ_χ fits. The flag is mandatory metadata in every spectrum artifact. |
| A-16 | Cubic (|z|²z) saturation is the lowest sufficient nonlinearity; coefficients β_nm ≥ 0. | M.46–M.47 | Hypothesis-grade assumption | If saturation is non-cubic (e.g., sin-bounded), steady-state amplitude vs power curves misfit; alternatives 2–3 of Module 10 are the registered fallbacks. |
| A-17 | Weak coupling for phase reduction: |K_nm| ≪ ω_n, and coupled modes are near-resonant (|ω_n − ω_m| ≪ ω_n) for the K_nm cos/sin form. | M.46–M.48 | Established (phase-reduction validity) | Far-detuned or strongly coupled modes need the full complex-coefficient coupling (rotating terms don't average out); fitted K_nm would be effective, not structural. |
| A-18 | The coherence window default (≥ 100 carrier cycles) balances temporal resolution and statistical robustness for RGCS bands. | M.56 | Derived (analysis choice; KOS trade-off) | Too short: baseline b_w inflates, coherence overestimated; too long: transitions smeared. Sensitivity analysis is mandatory; b_w measured per pipeline. |
| A-19 | Density is uniform at ρ = 2.65 g/cm³ (natural α-quartz), voids and inclusions negligible. | M.6, M.7, M.43 | Established (handbook value; citation required, D-19b) | Predicted-vs-measured mass check fails (RG-04 test consequence); density inverse returns biased dimensions; per-specimen measured mass overrides. |
| A-20 | Geometry is as modeled: regular polygonal sections, straight taper, symmetric facets, angle/diameter conventions correctly recorded per specimen. | M.1–M.7, M.39 | Established (metrology requirement) | Convention mix-ups (across-flats vs across-vertices; face_slope vs apex_included) produce percent-level volume errors and shift x_g; convention enums are mandatory metadata (RG-04, RG-21). |
| A-21 | Phase quantities are defined only relative to a declared shared reference clock (KOS-05 architecture); r_φ is computed on cycle counts, never on duty fractions. | M.48, M.50, M.53, M.58, M.61 | Established (measurement requirement; closes D-13) | Phase residues become pipeline artifacts (the WB5 duty-based residue bug produced 180° nonsense); golden test: half-spacing nominal 1507.328 cycles → r_φ = c − round(c) = +0.328 (+118.08°); the register's −0.328 is a sign erratum. |
| A-22 | Control configurations (dummy-load, no-crystal, far-detuned ε ≈ 1.25) are matched in energy, timing, and sensor chain to the active configuration. | M.57 | Established (design requirement) | G_c and d_c conflate configuration effects with instrument state; the exclusion logic of KOS-13 fails; claims revert to unadjudicated. |
